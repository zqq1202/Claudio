## 任务 1: MediaSession API（系统媒体控制）

在 `apps/web/src/audio/AudioPlayer.ts` 中集成 MediaSession API：
- 在 onPlay 回调中设置 `navigator.mediaSession.metadata`（需要 title, artist, artwork）
- 在 load(url) 后也更新 metadata（如果能从 playerStore 获取）
- 设置 action handlers: play, pause, previoustrack, nexttrack, seekto
- 播放/暂停通过已有的 audio.play()/audio.pause()
- prev/next 通过 playerStore 的 previous()/next()
- seekto 通过 audioPlayer.seek()

注意：AudioPlayer 不应该直接依赖 playerStore（避免循环依赖）。用回调注入的方式：
```typescript
setMediaCallbacks(callbacks: { next: () => void; previous: () => void; getTitle: () => string; getArtist: () => string; getCoverUrl: () => string })
```
然后在 playerStore 初始化时调用 `audioPlayer.setMediaCallbacks(...)` 注入。
