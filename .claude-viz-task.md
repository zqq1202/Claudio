## Task 2: Enhance SpectrumBars — Make Them Come Alive
File: `/projects/Claudio/apps/web/src/components/SpectrumBars.tsx`

Current problems:
- fftSize 256 (only 128 bins) → change to 1024 (512 bins) for more frequency resolution
- smoothingTimeConstant 0.8 → 0.7 (less built-in smoothing, more reactive)
- Attack 0.35 too slow → 0.5 (bars snap up fast)
- Release 0.12 too fast (bars disappear instantly) → 0.88 (bars float down slowly)
- Bar height `bars[i] * h * 1.3` → use logarithmic frequency mapping + perceptual bass boost
- No peak indicators → add peak dots that hold and fall
- No glow → add `ctx.shadowBlur` + `ctx.shadowColor` glow effect
- Height only 80px → increase to 120px (CSS change in global.css)
- Gradient: currently white only → use primary color gradient (bottom transparent → top bright)
- Add rounded top caps for modern look

Changes needed:

### A. Increase fftSize and use logarithmic mapping
```typescript
analyser.fftSize = 1024; // was 256
analyser.smoothingTimeConstant = 0.7; // was 0.8
```

### B. Better smoothing (attack/release)
```typescript
// Attack: 0.5 (fast rise), Release: 0.88 (slow fall)
bars[i] += (target - bars[i]) * (target > bars[i] ? 0.5 : 0.88);
```

### C. Logarithmic frequency mapping
Instead of `Math.pow(i / BAR_COUNT, 1.6)`, use proper log mapping:
```typescript
const minLog = Math.log10(1);
const maxLog = Math.log10(freqData.length);
const logIdx = Math.pow(10, minLog + (maxLog - minLog) * (i / BAR_COUNT));
const idx = Math.min(Math.floor(logIdx), freqData.length - 1);
```

### D. Perceptual bass boost
```typescript
const ratio = i / BAR_COUNT;
const boost = 1.0 + Math.pow(1.0 - ratio, 2.0) * 1.5; // boost low freqs
target = Math.min(1, (freqData[idx] || 0) / 255 * boost);
```

### E. Height scaling with sqrt compression
```typescript
const barHeight = Math.pow(bars[i], 0.7) * h * 0.9; // sqrt compression
```

### F. Peak indicators
```typescript
// Add peak tracking array
const peaksRef = useRef<number[]>(new Array(BAR_COUNT).fill(0));
const peakHoldRef = useRef<number[]>(new Array(BAR_COUNT).fill(0));

// In draw loop:
if (bars[i] > peaksRef.current[i]) {
  peaksRef.current[i] = bars[i];
  peakHoldRef.current[i] = 30; // hold 30 frames
} else if (peakHoldRef.current[i] > 0) {
  peakHoldRef.current[i]--;
} else {
  peaksRef.current[i] *= 0.97; // slow fall
}
// Draw peak dot
const peakY = h - Math.pow(peaksRef.current[i], 0.7) * h * 0.9;
ctx.fillStyle = primaryColor;
ctx.fillRect(x, peakY - 2, barWidth - gap, 2);
```

### G. Glow effect on bars
```typescript
ctx.shadowColor = primaryColor;
ctx.shadowBlur = 8 + bars[i] * 12;
```

### H. Better gradient
```typescript
const grad = ctx.createLinearGradient(x, h, x, y);
grad.addColorStop(0, toRgba(primaryColor, 0.05));
grad.addColorStop(0.3, toRgba(primaryColor, 0.15));
grad.addColorStop(0.7, toRgba(primaryColor, 0.4));
grad.addColorStop(1, toRgba(primaryColor, 0.8));
```

### I. Rounded top caps
```typescript
const radius = Math.min(3, (barWidth - gap) / 2);
ctx.beginPath();
ctx.roundRect(x, y, barWidth - gap, barHeight, [radius, radius, 0, 0]);
ctx.fill();
```
