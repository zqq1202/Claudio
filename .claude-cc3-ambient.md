# Task: Add Ambient Animation to Non-Player Pages

## Context
The Player page has visualizations (AudioVisualizer, SpectrumBars, particles) that make it feel alive.
But the other pages — Playlist, Profile, History, Settings — are completely static and feel dead.
Add subtle CSS-only ambient mesh gradient blobs as background decoration.

## Changes Required

### 1. global.css — Add Ambient Mesh Gradient CSS
File: `/projects/Claudio/apps/web/src/styles/global.css`

Add a NEW section at the end of the file (before any responsive media queries for mobile):

```css
/* ===== Ambient Mesh Gradient Background (non-player pages) ===== */
.ambient-mesh {
  position: fixed;
  inset: 0;
  overflow: hidden;
  pointer-events: none;
  z-index: -1;
}

.ambient-blob {
  position: absolute;
  border-radius: 50%;
  filter: blur(100px);
  opacity: 0.12;
  will-change: transform;
}

.ambient-blob-1 {
  width: 50vw;
  height: 50vw;
  top: -15%;
  left: -10%;
  background: radial-gradient(circle, var(--color-primary, #5ee8c5), transparent 70%);
  animation: ambientDrift1 20s ease-in-out infinite;
}

.ambient-blob-2 {
  width: 40vw;
  height: 40vw;
  bottom: -20%;
  right: -10%;
  background: radial-gradient(circle, #8b5cf6, transparent 70%);
  animation: ambientDrift2 25s ease-in-out infinite;
}

.ambient-blob-3 {
  width: 35vw;
  height: 35vw;
  top: 45%;
  left: 35%;
  background: radial-gradient(circle, #3b82f6, transparent 70%);
  animation: ambientDrift3 22s ease-in-out infinite;
}

@keyframes ambientDrift1 {
  0%, 100% { transform: translate(0, 0) scale(1); }
  33% { transform: translate(8%, 12%) scale(1.1); }
  66% { transform: translate(-4%, -8%) scale(0.95); }
}

@keyframes ambientDrift2 {
  0%, 100% { transform: translate(0, 0) scale(1); }
  33% { transform: translate(-12%, -8%) scale(1.05); }
  66% { transform: translate(8%, 4%) scale(1.08); }
}

@keyframes ambientDrift3 {
  0%, 100% { transform: translate(0, 0) scale(1); }
  50% { transform: translate(-10%, 15%) scale(1.15); }
}

/* Light theme — make blobs slightly more visible */
[data-theme="light"] .ambient-blob {
  opacity: 0.08;
}

@media (prefers-reduced-motion: reduce) {
  .ambient-blob {
    animation: none !important;
  }
}
```

For each of these 4 pages, add the ambient mesh div as the FIRST child inside the page's root element (the outermost `<div className="...-page">`):

```jsx

</div>
```
